# Order-MS(Project Status: In Progress 🚧)

A microservices-based backend project built with Java and Spring Boot, focused on distributed systems, asynchronous communication, and reliability patterns.

## Architecture & Concepts

* Microservices Architecture
* Event-Driven Communication
* Saga Choreography Pattern
* Retry & Dead Letter Queue (DLQ)
* Manual ACK/NACK Handling
* Idempotency Handling
* Inbox/Outbox Pattern
* Concurrency Handling with `FOR UPDATE SKIP LOCKED`

## Technologies

* Java
* Spring Boot
* Spring Security & JWT
* RabbitMQ
* PostgreSQL
* Redis
* Docker

## Services

* Order Service
* Payment Service

## Project Goals

This project is focused on gaining practical experience with:

* Distributed transactions
* Service-to-service communication
* Failure handling strategies
* Message reliability
* Scalable backend architecture
* Real-world microservice scenarios

## Current Status

The project is actively being developed and improved while exploring different distributed systems patterns and architectural approaches.
